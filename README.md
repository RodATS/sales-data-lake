# End-to-End Sales Data Lake

## 📌 Descripción
Proyecto de Data Engineering que implementa un Data Lake de ventas
con procesamiento batch y streaming usando Apache Spark y Apache Kafka.

## 🏗️ Arquitectura
Arquitectura por capas:
- Raw
- Bronze
- Silver
- Gold

## 🧰 Tecnologías
- Python
- Apache Spark
- Apache Kafka
- Hive
- Power BI

## 🎯 Objetivo
Demostrar habilidades prácticas de Data Engineering
en un pipeline end-to-end.
