## Dataset: Superstore Sales

Fuente: Dataset público de ventas tipo retail (Superstore)

Descripción:
Contiene registros de órdenes de venta con información de clientes,
productos, fechas, ubicación y monto de ventas.

Columnas principales:
- Order_ID: identificador de la orden
- Order_Date: fecha de la orden
- Ship_Date: fecha de envío
- Customer_ID: cliente
- Segment: segmento de cliente
- Product_ID: producto
- Category / Sub_Category
- Sales: monto de venta
- City / State / Region
